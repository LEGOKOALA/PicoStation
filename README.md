# Pico Station
You are trapped in a space station, and you must cooperate with your compatriats in order to fix the ship and save yourselves! But beware, the shaky gravity has made it easy for your senses to be turned upside down!

Requirements from a hat:
1. Cooldowns - Try to spam any ability, is in player_movement scripts
2. Co-op - Connect more than one controller, is the player 1-4 nodes
3. Environment - Make it to lvl 2, is part of the players and lvl 2
4. Gravity Inversion - Press X, is in player_movement scripts

OOP and Godot Features:
1. 'is': can be found in the Main script
2. enum: can be found in the player_movement scripts
3. generic instantiation: can be found in the main script
4. abstract class: can be found in player1_movement script
5. private methods/variables: can be found in the player_movement scripts
6. use a PackedScene: can be found in the main script
7. _Ready function: can be found in the player_movement scripts
8. Custom inputs: can be found in the input map under project settings
9. Changes scenes: changes scenes when switching levels
10. Uses a custom signal: can be found in the darknesstrigger script
